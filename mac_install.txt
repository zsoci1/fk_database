1. install python  https://www.python.org/downloads/macos/

2. cd ~/Documents - open terminal and navigate to the project folder

3. create virtual environment
    cd fitkitchen_database
    python3 -m venv venv
    source venv/bin/activate

4. install dependencies
    pip install -r requirements.txt

5. run the app
    python main.py

6. create a mac shortcut
    open automator.app
    search for automator using spotlight
    choose application type

    add a "Run Shell Script" block
    paste this:
    cd ~/Documents/fitkitchen_database
    source venv/bin/activate
    python ui/main.py

    save it 
    Cmd + S 
    name it Fitkithcen App 
    choose Application folder or Desktop
